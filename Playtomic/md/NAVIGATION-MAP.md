# Playtomic Navigation Map

## Root Navigation

Authenticated bottom navigation appears to be:

```text
Home | Community | Profile
```

Each root tab also uses a shared blue top shell with:

- notification bell
- hamburger/menu icon

## Public / Pre-Auth

```text
Login
  -> Continue with Google
  -> Continue with Facebook
  -> Continue with email
```

## Search / Booking Sub-Navigation

```text
Book a court
  -> Search location
  -> View map
  -> Filter panel
     -> Duration
     -> Type
     -> Features
     -> Size
  -> Sport selector
  -> Date selector
  -> Time selector
```

## Important Navigation Observations

- Home is the clear product launchpad.
- Booking is a deep branch from Home rather than its own persistent tab.
- Community is meaningful but secondary to court search.
- Premium and progression cues live close to Profile.
