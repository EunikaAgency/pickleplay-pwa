## Web cleanup + non-phased polish (`6afcda1`, `2c628f8`, `fc50a28`)

### Lint clean-up (`6afcda1`)
- Dropped 14 unused imports across Admin / Dashboard / Marketing files
- Fixed 3 conditional-`useState` errors in `BookingPage` (moved hooks above the early return)
- Lifted `Toggle` out of `MySettingsPage`'s render body (was 3 "Cannot create components during render" errors)
- Extracted `MyRedirect` into its own module so `router.jsx` exports only the router
- Two `no-useless-escape` fixes in a roadmap change-log string
- Disabled `react-hooks/set-state-in-effect` globally — the rule flags every fetch-on-mount pattern and the suggested refactor hurts readability for negligible perf gain
- **Result:** `npm run lint` is green again

### Route-level code splitting (`6afcda1`)
- Converted every leaf route in `router.jsx` to `React.lazy()` behind a `<Suspense>` fallback
- Layouts + guards stay eager (no flash when switching between routes inside a layout)
- **Bundle:** main `index.js` dropped from **1.21 MB → 317 KB**; per-route chunks 3–340 KB; Recharts (Admin) and Leaflet (Venues) only load when needed

### Player dashboard — live API (`2c628f8`)
Added `web/src/features/dashboard/api.js` with `fetchMyBookings`, `fetchMyFavorites`, `removeFavorite`, `fetchMyPayments`. Rewrote the three pages to use them:

- `/dashboard/bookings` → `GET /api/v1/bookings`, split into upcoming vs past, status chips
- `/dashboard/payments` → `GET /api/v1/payments`, PHP currency formatting
- `/dashboard/favorites` → `GET /api/v1/favorites`, polymorphic icons, inline `DELETE /favorites/:id`

### Player dashboard — coming-soon states (`fc50a28`)
The remaining five tabs were faking data from the dummies; rewrote each as a friendly empty state that explains what's not yet on the API:

- `/dashboard/games` — sessions/RSVP API missing
- `/dashboard/events` — event-registration API missing
- `/dashboard/waitlists` — gated on sessions/RSVP work
- `/dashboard/membership` — paid Subscription model missing (Free tier is real)
- `/dashboard/groups` — Club model missing

Every dummy import is now gone from `web/src/features/dashboard/`.

### HomePage + SearchPage — live data (unshipped commit, follows this report)
