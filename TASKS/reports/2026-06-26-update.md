PickleBallers — Update (June 26)
For Venue Owners

Split-Court / Half-Court Booking

•  Owners can now mark a court as splittable and define how many half-courts it divides into (2, 3, or 4 sub-units).
•  Each sub-unit can have its own hourly rate; if left blank, the court's main rate applies.
•  On the booking flow, players see a "Court unit" picker (Full court / Half 1 / Half 2 / …) when the chosen court is splittable.
•  The pricing engine resolves sub-unit rates correctly: sub-unit rate → court rate → venue rate.
•  Sub-unit booking is clash-enforced — a half-court booking blocks that sub-unit only, not the whole court.
•  The public court page now shows "Splittable into N half-courts" with per-unit rates when set.
•  Done: Yes
•  Tested: For QA only

Equipment Rental Add-ons

•  Owners can set an equipment/paddle rental price per venue in the Listing tab.
•  Players see an "Add equipment rental" toggle in the booking flow below the court picker.
•  When toggled on, the rental amount is added to the subtotal and shown as its own line item in the review step.
•  The rental flag and amount are persisted on the booking and visible on My Bookings and the owner booking detail.
•  Done: Yes
•  Tested: For QA only

Time-Block Pricing Applied During Booking

•  If an owner sets a different hourly rate for a specific time window (e.g. 6 PM–10 PM = ₱500/hr), that rate now correctly applies when a player books within that schedule.
•  The booking flow fetches the venue's operating hours and resolves the matching time block for the chosen day and start time.
•  Rate precedence: time-block rate → sub-unit rate → court rate → venue rate.
•  Cleaned up the WeeklyHoursEditor layout so the time fields, price input, and remove button align properly on mobile instead of wrapping awkwardly.
•  Done: Yes
•  Tested: For QA only

Cancellation & Refund Policy (Owner-Configurable)

•  Owners can now set cancellation rules per venue in the Listing tab.
•  Fields: cancellation window (hours before booking), refund percent, and no-show fee.
•  The policy is shown to players on the public court page and in the booking review step before they commit.
•  The platform enforces these rules automatically (refund calculation, no-show charges).
•  Done: Yes
•  Tested: For QA only

Court Profile Fields

•  Each court now has physical-attribute fields in the Courts tab: Floor type (Wood / Professional), Ball type (Indoor / Outdoor), Air-conditioned, High ceiling, Refreshment stand, and Space around court (free-text).
•  These render as descriptive chips on the public court page (e.g. "Wood floor · Indoor ball · High ceiling · 3m clearance").
•  Done: Yes
•  Tested: For QA only

Desktop Sidebar for Admins & Owners

•  Admin and owner roles now get a full sidebar layout on desktop (≥1024px) instead of the centred phone-width frame.
•  Player, coach, and organizer roles keep the mobile phone-frame layout on all screen sizes.
•  Done: Yes
•  Tested: For QA only

Pricing Tax Label

•  Owners can customize the tax-disclosure label shown on the public page and checkout review (e.g. "VAT inclusive", "Tax included").
•  Defaults to "VAT inclusive" when not set.
•  Done: Yes
•  Tested: For QA only
