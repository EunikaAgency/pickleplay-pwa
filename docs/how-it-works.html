Then, depending on the type and the open/private toggle…

🎾Hosted game
Game posted with a lobby

After paying, tap View your game. Others find it and join up to your slot cap; everyone shares a group chat.

👥Open play session
Opens as a drop-in or stays private

The Who can join toggle decides: 🌐 Open play publishes the session for drop-ins; 🔒 Private game keeps it as a plain court reservation — nothing is published, nobody else can join.

⏳
Some venues approve first. If the owner requires it, the last step becomes Request booking — your card is saved but not charged. The owner approves, you get a notification, then you pay within the time window to lock it in.

The whole thing, in four lines

If you remember nothing else, remember this.

Owner setup
  1. Add venue (name + map pin)
  2. Add courts + base rate
  3. Paint prices on /owner/pricing
  4. Save → it's bookable
Book a court
  1. Home → Book Court
  2. Pick a place → booking type
  3. Details (open/private toggle)
  4. Date, time, pay → confirmed
Join a hosted game
  1. Home → Play
  2. Events tab → open a game
  3. Join the lobby
  4. Chat + show up
Join open play
  1. Home → Play
  2. Open Play tab → a session
  3. Join (or accept an invite)
  4. Just show up

PickleBallers — owners fill their courts, players find their game.
