# app/ — PickleBallers PWA

Mobile-first React PWA for finding pickleball games, courts, clubs, and players.

- 🗺️ **The file-map is [FILEMAP.md](FILEMAP.md)** — skim it first before
  grepping/scanning `src/`. It points you straight at the right file.
- Deep architecture + the add-a-screen checklist live in [CLAUDE.md](CLAUDE.md).
- Repo-wide conventions for all agents live in [../AGENTS.md](../AGENTS.md).

Quick start:

```sh
npm install
npm run dev      # Vite dev server on port 9000 (API proxied to :9002)
```
