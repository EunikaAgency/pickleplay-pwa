21 July 2026

✓ Done & working

Asking to join now works — and you can back out

Before, if you asked to join and left the screen, the app forgot and tapping again threw an error. Now it stays “waiting for the host”, and a Cancel request button lets you change your mind.

Check it yourself: open an approval play → tap Request to join → close and reopen it. It still shows “waiting” with a Cancel button.
✓ Done & working

Hosts can finally edit their play

After posting a play, the host can now change the name, skill, player limit, description, visibility, and the “approve players first” switch — no more delete-and-start-over.

Check it yourself: open a play you host → tap Edit lobby → change anything → Save changes.
✓ Done & working

Staff can no longer see the money

This revenue view is for owners only now. A staff login sees the same screens with the money removed everywhere — exactly what the meeting asked for.

How we know: the revenue permission was taken away from staff, and an automated test locks it in so it can't come back by accident.
