# ReClub Navigation Map

## Root Navigation Model

ReClub appears to use a tile-based dashboard instead of a classic bottom-tab structure.

Visible home/dashboard tiles:

```text
My clubs | My network | My history
Feed | Statistics | Settings
Help | Create | See less
```

Visible top-right utility icons on several home-like screens:

- notifications bell
- chat/inbox bubble

## Onboarding / Entry

```text
Get started
  -> Sign up front
  -> User signup input
  -> Select sports and level
  -> Successful signup
  -> Home dashboard
```

## Search Overlay Navigation

The global discovery/search sheet appears to be a second navigation layer.

Tabs visible in the overlay:

```text
Clubs | Meets | Comps | Players | Coaches
```

Filter controls visible:

- location dropdown
- `Your sports` dropdown
- community filter
- map filter
- map toggle or map mode
- search field

## Important Navigation Observations

- Discovery is centralized in one large searchable overlay.
- Clubs are both public-discovery objects and persistent private/member spaces.
- Creation is a core first-class feature, not a hidden admin tool.
- The app separates informal `meets` from structured `competitions`.
